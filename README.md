# Polytec Lab website

Product site for Polytec Lab — analog-inspired music software.

## Stack

- **Next.js 14** — App Router
- **TypeScript**
- **Tailwind CSS**
- Fonts: Syne (display), IBM Plex Mono, Crimson Pro — all via `next/font/google`

## Getting started

```bash
npm install
npm run dev
# → http://localhost:3000
```
